# Young & Hungry

Young & Hungry is a static website for college students learning how to use AI more effectively and responsibly. The site is free, self-paced, and designed to work well on desktop and mobile.

## What's Included

- `index.html` redirects visitors into the site experience.
- `home.html` is the main landing page.
- `responsible.html` covers responsible and unethical AI use.
- `topic-resume.html`, `topic-interview.html`, and `topic-career.html` focus on career-related AI skills.
- `time-management-prod.html` covers time management.

## Run Locally

Open the site in a browser by launching `index.html`.

```bash
open index.html
```
